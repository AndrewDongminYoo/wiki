# How to SignIn Git Credential

Git 자격 증명에 로그인하려면 다음 단계를 따르세요:

1. 컴퓨터에서 (Git) Bash 또는 명령 프롬프트를 엽니다.
2. `git config --global credential.helper store`를 입력하고 Enter를 누릅니다. 로그인 자격 증명을 저장하도록 GIT가 구성됩니다.
3. `git config --global user.name`를 입력하고 Enter를 누릅니다. GIT 사용자 이름으로 교체하십시오.
4. `git config --global user.email`를 입력하고 Enter를 누릅니다. GIT 계정과 관련된 이메일 주소로 교체하십시오.
5. `git config --global credential.username`를 입력하고 Enter를 누릅니다. 이렇게하면 GIT 사용자 이름을 GIT 자격 증명의 기본 사용자 이름으로 설정합니다.
6. `git config --global credential.helper 'cache --timeout=3600'`를 입력하고 Enter를 누릅니다. 이로 인해 GIT 자격 증명에 대한 시간 초과 1 시간이 설정됩니다.

이 단계를 완료하면 Git 사용자 이름과 비밀번호를 사용하여 Git 자격 증명에 로그인할 수 있습니다.
